# Project 3 - Web App and Application Server ## Introduction This project is a simplified online community and content management system that allows user to create threads or post articles and leave comments below others' threads or articles. ## Structure A **Forum** will include multiple **Categories**, which are independent discussion sections about specific subtopic. In each Section, there will be multiple **Threads** posted by users, and also the **Posts** below each posts replied by other users. Each user may have there own **Personal Page** showing his/her activities on this forum ## Features - Markdown based post and comment content editing - A **Portal** page as the home page that provides a direct way for users to see latest or recommended posts ## Link to the project **Please launch the django server first before open the link** [https://mac1xa3.ca/e/xiny13/](https://mac1xa3.ca/e/xiny13/)
